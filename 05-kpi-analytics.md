# KPI и аналитика

## 1) KPI на 90 дней
- Индексация целевых страниц первой волны: не ниже `85%`.
- Видимость по высокоприоритетным кластерам: рост по неделям.
- Количество заявок с SEO-трафика: базовая конверсия в заявку `>= 2%`.
- CTR из поиска по коммерческим кластерам: рост после оптимизации сниппетов.

## 2) KPI на 180 дней
- Расширение индексируемого пула до волн 2-3 без резкого роста дублей.
- Стабильный прирост органического трафика месяц к месяцу.
- Рост доли бренд+небренд запросов в пользу небренд коммерческих ключей.

## 3) Обязательные подключения
- Яндекс Метрика.
- Google Analytics 4.
- Google Search Console.
- Яндекс Вебмастер.

## 4) События и цели (минимальный набор)
- `lead_submit`: отправка формы.
- `lead_click_phone`: клик по телефону.
- `lead_click_messenger`: клик в мессенджер.
- `lead_click_cta_primary`: клик по главному CTA.
- `scroll_75`: скролл до 75% страницы (качество контента).

## 5) Сегментация отчетов
- По типу страницы: услуга / услуга+ниша / услуга+ниша+город.
- По услуге.
- По нише.
- По городу.
- По источнику: Yandex / Google / referral / direct.

## 6) Операционный ритм
- Еженедельно: индексирование, технические ошибки, страницы без трафика.
- Раз в 2 недели: корректировка title/description и перелинковки.
- Ежемесячно: пересборка приоритетов и запуск новой пачки кластеров.
