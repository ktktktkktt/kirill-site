# Контент-брифы для шаблонов

## 1) Шаблон: услуга (`/service/{serviceSlug}`)
- H1: `Разработка {serviceName} под ключ`
- Первый экран: оффер + срок + цена от + CTA.
- Блоки:
  - кому подходит услуга,
  - что входит,
  - сроки и этапы,
  - стоимость (3 пакета),
  - кейсы,
  - FAQ,
  - финальный CTA.
- Минимум текста: `2500+` символов.

## 2) Шаблон: услуга + ниша (`/service/{serviceSlug}/{nicheSlug}`)
- H1: `{serviceName} для {nicheName}`
- Уникальные блоки:
  - боли ниши,
  - обязательные функции для ниши,
  - примеры структуры страницы,
  - кейс по этой нише или аналог.
- Минимум текста: `3000+` символов.

## 3) Шаблон: услуга + ниша + город (`/service/{serviceSlug}/{nicheSlug}/{citySlug}`)
- H1: `{serviceName} для {nicheName} в {cityName}`
- Геомодификаторы:
  - локальный спрос,
  - локальная конкуренция,
  - локальные формулировки CTA.
- Локальные элементы:
  - блок "работаем с {cityName}",
  - преимущества по срокам/процессу удаленной работы,
  - FAQ с упоминанием города.
- Минимум текста: `2200+` символов.

## 4) Шаблонные переменные для генерации
- `serviceName`, `serviceSlug`
- `nicheName`, `nicheSlug`
- `cityName`, `cityRod`, `citySlug`
- `priceFrom`, `timeline`, `ctaText`
- `nichePainPoints[]`, `nicheFeatures[]`, `faqItems[]`

## 5) Правила уникализации массовых страниц
- Не менее `35%` текста страницы должно быть уникальным для связки услуга+ниша+город.
- Уникальные H1/H2 и минимум 3 уникальных абзаца.
- Уникальный FAQ (минимум 3 вопроса).
- Уникальные meta title и description.
