# Семантическое ядро и приоритизация

## 1) Услуги (основа структуры)
- `odnostranichnyj-sajt`: одностраничный сайт
- `mnogostranichnyj-sajt`: многостраничный сайт
- `internet-magazin`: интернет-магазин
- `brendirovanie`: брендирование
- `upakovka-pod-klyuch`: упаковка под ключ
- `redizajn`: редизайн
- `po-zaprosu`: по запросу

## 2) Ниши (целевой набор)
- `stroitelstvo-domov`: строительство домов
- `remont-kvartir-domov`: ремонт квартир и домов
- `landshaftnyj-dizajn`: обустройство участков и ландшафтный дизайн
- `zastrojshchiki`: застройщики
- `remont-bytovoj-tekhniki`: ремонт бытовой техники
- `avtotematika`: автотематика
- `psikhologi`: психологи
- `drugoe`: другое

## 3) Города
- `moskva`
- `spb`
- `ekaterinburg`
- `kazan`
- `krasnodar`
- `novosibirsk`
- `nizhnij-novgorod`
- `rostov-na-donu`

## 4) Кластеры запросов

### Кластер A: услуга (высокий приоритет)
- разработка одностраничного сайта
- заказать одностраничный сайт под ключ
- стоимость многостраничного сайта
- разработка интернет-магазина цена
- создать интернет-магазин под ключ

### Кластер B: услуга + ниша (высокий приоритет)
- сайт для строительной компании
- сайт для ремонта квартир
- сайт для застройщика
- интернет-магазин для автотоваров
- сайт для психолога

### Кластер C: услуга + город (средний приоритет)
- разработка одностраничного сайта москва
- многостраничный сайт спб под ключ
- создание интернет-магазина екатеринбург
- редизайн сайта казань
- сделать сайт краснодар

### Кластер D: услуга + ниша + город (высокий приоритет, массовые посадочные)
- сайт для строительной компании в москве
- сайт для ремонта квартир в спб
- интернет-магазин для автотематики в екатеринбурге
- сайт для психолога в казани
- редизайн сайта для застройщика в краснодаре

## 5) Логика приоритета
- **Высокий**: коммерческий интент + высокая частотность + явный формат услуги.
- **Средний**: коммерческий интент есть, но часть запросов шире по смыслу.
- **Низкий**: информационные или слишком размытые формулировки.

## 6) Волна 1 (для старта разработки)
- Услуги: `odnostranichnyj-sajt`, `mnogostranichnyj-sajt`, `internet-magazin`.
- Ниши: `stroitelstvo-domov`, `remont-kvartir-domov`, `psikhologi`.
- Города: `moskva`, `spb`, `ekaterinburg`, `kazan`, `krasnodar`.

Объем волны 1 для связки услуга + ниша + город: `3 x 3 x 5 = 45` страниц.
